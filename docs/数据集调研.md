# 数据集调研报告

## 1. 主流跌倒检测数据集

### UP-Fall Detection Dataset
- **论文**: UP-Fall: A Dataset for Fall Detection (Sensors, 2019)
- **规模**: 17 名被试, 11 类动作 (5 ADL + 6 跌倒)
- **传感器**: RGB + Depth + IR + 加速度计
- **帧率**: 30 FPS
- **特点**: 最全面的多模态数据集，有时间序列标注
- **获取**: 免费申请

### Le2i Fall Detection Dataset
- **来源**: Laboratoire Le2i, Burgundy University
- **规模**: 4 个室内场景
- **传感器**: RGB
- **帧率**: 25 FPS
- **特点**: 最经典、引用最多
- **获取**: 免费下载

### NTU RGB+D
- **规模**: 56,880 视频, 60 类动作
- **传感器**: RGB + Depth + IR
- **特点**: 规模巨大，含跌倒类，适合泛化测试
- **获取**: 免费申请

### Multicam Fall Dataset
- **特点**: 8 个摄像头同步，多视角
- **用途**: 多视角跌倒检测研究

## 2. 数据集特点分析

### 2.1 类别不平衡

跌倒事件占比通常 < 5%。采用 Focal Loss 和加权采样处理。

### 2.2 老人 vs 年轻人

大部分数据集用年轻人模拟。需要域迁移/少样本适配。

### 2.3 标注粒度

- 二分类: 跌倒/非跌倒
- 多阶段: 前倾→失衡→跌倒→躺地

### 2.4 多模态趋势

最新论文（2025-2026）都在做多模态融合。

## 3. 本项目数据集策略

1. **UP-Fall 为主** — 多模态、时间序列标注
2. **Le2i 补充** — 经典基准、方便对比
3. **NTU RGB+D 泛化** — 证明模型通用性
