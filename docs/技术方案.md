# 技术方案

## 1. 项目概述

**题目**: 基于多模态AI监测的老年人跌倒风险、心理健康、诈骗识别及预警研究
**方向**: 跌倒风险前置预判
**发榜单位**: 海康威视 / 萤石

## 2. 创新点

### 2.1 连续性风险评分模型

不同于传统的二分类（跌倒/非跌倒），本系统输出 0-100 的连续风险评分，提供更细粒度的风险评估。

### 2.2 多维度融合预警

融合萤石平台多个 API 输出（人体检测、姿态分析、步态分析），构建综合风险评估模型。

### 2.3 个性化基线

每位老人建立独立的正常状态基线，通过检测偏离程度来评估风险，避免"一刀切"的阈值判断。

### 2.4 分级预警 + 联动响应

- 低风险 (0-30): 继续监控
- 中风险 (31-60): 提高监控频率
- 高风险 (61-80): 通知家属
- 极高风险 (81-100): 紧急响应

## 3. 技术架构

```
摄像头取流 (萤石)
    ↓
人体检测 (YOLOv8 / 萤石API)
    ↓
姿态估计 (MediaPipe / HRNet)
    ↓
步态分析 (LSTM / Transformer)
    ↓
多模态特征融合 (Attention Fusion)
    ↓
风险评分模型 (MLP + 校准)
    ↓
分级预警 → 通知推送
```

## 4. 模型设计

### 4.1 姿态估计

- MediaPipe Pose: 33 关键点，映射到 COCO 17 点
- 实时性能: CPU 15-20 FPS, GPU 30+ FPS

### 4.2 步态分析

- LSTM: 双向 LSTM + 注意力池化，适合小数据集
- Transformer: 自注意力 + CLS token，适合大数据集

### 4.3 风险评分

- 输入: 多模态融合特征
- 输出: [0, 100] 连续评分
- 损失: MSE + 排序损失
