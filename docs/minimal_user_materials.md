# 你后续只需要提供的最小材料

你不用写代码，也不用改模型。后面最需要你补的是这些“比赛现实材料”：

## 第一组：真实测试证据

- 1 到 3 段居家场景视频
- 或一组居家场景截图与说明
- 每段样例说明发生在什么场景：客厅、卧室、卫生间、走廊

## 第二组：平台接入证据

- 萤石开放平台账号信息中的非敏感部分截图
- 设备列表、接口调用结果、平台控制台页面截图
- 如果没有设备，也可以先提供你能拿到的平台页面截图或沟通记录

## 第三组：报名与身份信息

- 学校名称
- 团队名称
- 负责人姓名
- 手机号
- 指导老师信息

## 第四组：最终报告素材

- 你希望强调的创新点
- 你们团队的真实分工
- 是否已有和老师、同学讨论过的实验结论

## 你不需要做的事

- 不需要自己改代码
- 不需要自己配环境
- 不需要自己写技术报告
- 不需要自己整理指标口径

你只需要把上面这些现实材料陆续给我，我来把它们编进项目、报告和演示里。
